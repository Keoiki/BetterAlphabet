Folder of scripts (known as "Commands" in Adobe Animate) for use in Animate.

### AutomateSymbolCreationWithGrid
Creates symbols with numbers as their name. The amount of symbols depends on the document height/width divided by the set horizontal/vertical divisions.

### ExportItemsAsSVG
Exports all items in the Library as SVG files. Location is relative to where the .fla file is located.
The base export folder is the same name as the .fla file. (latin.fla -> latin/)
>(Folders must be created manually, as the script will not create them.)
